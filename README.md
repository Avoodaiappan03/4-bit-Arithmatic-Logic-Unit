# 4-bit Arithmetic Logic Unit (ALU)

A professional Verilog HDL implementation of a 4-bit ALU, developed as part of my VLSI Verification portfolio.

## Project Features
- **Data Width:** 4-bit
- **Operations:** Supports Addition, Subtraction, AND, OR, XOR, and NOT.
- **Flags:** Includes a Carry-Out flag for arithmetic operations.
- **Verification:** Includes a comprehensive testbench for functional simulation.

## File Descriptions
- `alu.v`: RTL Design source code.
- `alu_tb.v`: Testbench used for functional verification.
- `waveform.png`: (Optional) Screenshot of simulation results.

## How to Simulate
1. Open [EDA Playground](https://edaplayground.com).
2. Copy the `alu.v` and `alu_tb.v` code into the respective windows.
3. Select "Icarus Verilog" as the tool and click "Run."
